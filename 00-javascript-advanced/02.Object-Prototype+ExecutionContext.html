Pembelajaran Tentang Prototype:

  1. Prototype itu sudah ada di bawaan default javascript contohnya di setiap object, array, bahkan number pun sebenarnya sudah ada
  2. Sebelum dibungkus kedalam class (OOP), sebenarnya sistem yang berjalan dibelakang layar terdiri dari object dan juga prototype nya
  3. tes object : kiikun

Pembelajaran Tentang Execution Context , Hoisting, Scope

  1. Pada execution context javascript dimulai dengan 2 tahap yaitu creation phase (terdapat cara hoisting) dan execution phase
  2. [Creation] cara kerja hoisting diawali dari penarikan data semua nama object dengan nilai undefined dan penarikan nama2 function beserta isinya
  3. [Execution] mengeksekusi code program baris per baris dari atas sampai bawah
  4. Scope terdiri dari Global dan Local
  5. [Global Scope] object yang di define global (diluar function)
  6. [Local Scope] object yang di define local (didalam function)
